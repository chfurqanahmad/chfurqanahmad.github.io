## Why this site is lightweight

The website does not require Node.js, Jekyll, Ruby, npm or a local build process. GitHub Pages can serve the files directly.

## Upload through the GitHub website

1. Open the `chfurqanahmad.github.io` repository.
2. Download or preserve the current `index.html` if you want a backup of the Coming Soon page.
3. Upload all files and folders from this package into the repository root.
4. Confirm that the `CNAME` file still contains only:
   `furqanahmad.com`
5. Commit the changes to `main`.
6. Wait for GitHub Pages to redeploy.
7. Test `https://furqanahmad.com` on desktop and mobile.
